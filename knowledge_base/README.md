# Knowledge Base Directory

Place all lending policies, guidelines, and reference documents here.
The RAG system will split, embed, and index these files to retrieve policy clauses and guidelines.

## Expected Documents
1. **Internal Credit Policy**: Lending terms, interest rates, eligibility criteria.
2. **RBI Guidelines**: KYC (Know Your Customer) rules, lending practices, Digital Lending guidelines.
3. **Fraud Policies**: Criteria to identify or escalate potential fraudulent applications.
